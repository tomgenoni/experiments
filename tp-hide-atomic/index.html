Replace Layout with Atomic

As part of the effort to migrate deprecated tp utility classes to Atomic, I put together this conversion chart.

In the left column are the deprecated utility classes along with the number of found instances in website and on the right are the Atomic equivalents. As you resize your browser the solid gray boxes will show and hide to demonstrate that the deprecated utility classes and the Atomic replacements are operating as expected.

The examples outlined in red that use a db class could instead be flex, inline-flex, di, di, dt, or dib. This is because the deprecated classes only hid in specific range and left the display value to another class or in the Sass. Each instance of this would require manual conversion.
